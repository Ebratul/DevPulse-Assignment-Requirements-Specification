# Express Assignment 🚀

A backend web application built with **Node.js, Express, TypeScript, and Prisma**, deployed on **Vercel**.

---

## 🌐 Live URL

👉 https://espressassignment.vercel.app/

---

## ✨ Features

- RESTful API architecture
- TypeScript-based backend for type safety
- Express.js server setup
- Prisma ORM integration for database handling
- Environment variable configuration using `.env`
- Modular folder structure (controllers, services, routes)
- Production deployment on Vercel

---

## 🛠️ Tech Stack

- Node.js
- Express.js
- TypeScript
- Prisma ORM
- PostgreSQL / MySQL (depending on your DB)
- Vercel (Deployment)
